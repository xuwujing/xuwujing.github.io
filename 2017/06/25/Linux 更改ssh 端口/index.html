Linux 更改ssh 端口

发表于 2017-06-25

部署了一个测试服务器之后，在查看linux日志的时候，发现莫名的IP一直在访问服务器，感觉像是某种恶意扫描，来攻击服务器的。因此更改ssh端口。

输入: vim /etc/ssh/sshd_config
找到port 并解除注释，更改端口
或者不动注释，新建个port

重启 ssh 输入: service sshd restart
然后在重新通过ssh连接，将22端口改为设置的端口，就可以登录了。
