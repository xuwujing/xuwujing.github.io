Web 项目更改项目名

发表于 2017-06-25 | 分类于 开发工具

简单的记录web开发中基本的操作。

更改项目名

直接修改

找到原项目中的.project 文件，更改中项目名称。然后在同目录下找到.mymetadata 文件 并更改name、context-root、archive 中的项目名称。这些名称必须一致。重新导入，项目名称就成功更改了。

使用eclipse修改

已导入eclipse中的项目，首先点击F2进行重命名，然后右键点击该项目，选择properties，然后搜索web，更改Web Context-root中的名称，点击保存之后。项目名称更改就成功了。
