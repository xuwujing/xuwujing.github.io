微软云Linux服务器 Mysql、tomcat远程连接错误解决办法

发表于 2017-04-28 | 分类于 linux

问题

在微软云linux服务器成功配置好mysql、tomcat，通过外部链接却发现一直错误。Mysql 一直提示错误代码2003，


tomcat连接一直提示EOF。

解决

反复检查配置都无问题，最后得知是微软云linux服务器默认的端口都是关闭的，需手动打开。成功打开之后就可以链接了。
