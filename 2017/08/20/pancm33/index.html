Jquery一些常用的方法

发表于 2017-08-20 | 分类于 javascript

整理以前的笔记，在学习JavaScript时候,经常会用到一些方法，但是有时忘掉了具体用法，因此记下。方便以后查阅。
这篇博文先说明这些方法的用途：
removeClass()、remove()、css() 、prop() 、html()、appendTo() 、clone() 、parent() 、children()、nth-child(n)、even、odd 。

.addClass() 增加样式

例如: $(“button”).addClass(“animated bounce”); /按钮做弹回效果 /

.removeClass() 移除样式

例如: $(“button”).removeClass(“btn-default”); /移除样式 /

.remove() 移除元素

例如:$("#target4").remove();   //移除按钮四所有的元素

.css() 改变css样式

例如: $("#target1").css("color", "red"); //按钮一改成了红色

.prop() 调整元素属性

例如:$("#target1").prop("disabled",true); //按钮一不可选

.html() 可以添加HTML标签和文字并替换掉之前的内容
例如:$(“#target4”).html(“#TarGet4“); //替换并增强文本

.appendTo() 可以将选中的元素移到其他元素中

例如: $("#target2").appendTo("#right-well");  //将按钮二移动到右边

.clone() 拷贝元素

例如:$("#target5").clone().appendTo("#left-well");  //将按钮五拷贝到左边

.parent() 访问父元素(上一级元素)

例如:$("#target1").parent().css("background-color","red");//按钮一上一级的 背景颜色为红色

.children() 子元素(下一级元素)

例如: $("#right-well").children().css("color","orange"); //右边一下的元素都改成了橙色

.nth-child(n) 通过CSS选择器索引顺序选择目标元素的子元素

例如:$(".target:nth-child(2)").addClass("animated bounce"); //class 属性包含target 是添加元素

.even 偶数选择

例如: $(".target:even").addClass("animated shake");  //获取class为target且索引为偶数的所有元素

.odd 奇数选择

例如:$(".target:odd").addClass("animated bounce");
