学习网站

freecodecamp 一个非常好的网站，教学模式类似游戏中的闯关，通过每关之后会有成就感，在该网站还有设有聊天室，可以进行相关的技术交流，很棒的学习网站。
https://www.freecodecamp.cn/

菜鸟教程 对待初学者很友好，类似一个demo基础的网站。
http://www.runoob.com/

w3c 这个网站不必多说了
http://www.w3school.com.cn/

并发编程网 有一定的基础之后再去学习，程序员的进阶的网站
http://ifeve.com/

Github 全球最大的开源社区，三个字，强无敌！
https://github.com/

Stack Overflow 国外的技术交流网站 ，很多技术问题可以在其中找到。
https://stackoverflow.com/
